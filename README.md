# 📝 Django Blog Management System

A full-stack blog management web application built with **Django**, allowing users to browse blogs, search posts, comment on articles, and manage content through a secure dashboard. The application features authentication, role-based authorization, category management, and a responsive Bootstrap interface.

---

## 🚀 Features

### User Features

- User Registration
- User Login & Logout
- Browse Published Blogs
- Search Blogs
- View Blogs by Category
- Post Comments
- Responsive User Interface

### Admin / Manager Features

- Dashboard Overview
- Create Blog Posts
- Update Blog Posts
- Delete Blog Posts
- Manage Categories
- Manage Users
- Role-Based Authorization (Manager Access)

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/aditya123-456/django-blog.git
```

### Navigate to Project

```bash
cd django-blog/blog_main
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Apply Database Migrations

```bash
python manage.py migrate
```

### Run Development Server

```bash
python manage.py runserver
```

Visit:

```
http://127.0.0.1:8000/
```
